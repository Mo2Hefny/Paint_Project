# Paint
## Description
- This is a simple Paint For Kids application that replicates some functionalities of Windows Paint.
- This program was written entirely in C++ as a project for our first year in CMP.
## Installation
1. Clone this repository.
2. Open the `.sln` file.
3. Build solution.
